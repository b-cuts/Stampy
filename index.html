Stampy - a Java Implementation of the STOMP 1.2 Specification

This is a placeholder for a high-level description. Stampy creates STOMP messages via Java classes and uses Apache MINA to facilitate network connections. STOMP messages are dealt with via an implementation of an interface. It is simple to use and requires little configuration, while leveraging MINA to allow many simultaneous client connections. It includes both a client & a server implementation which can easily be tailored for individual use.

A work in progress - a release will be made available shortly.
